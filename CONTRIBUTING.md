# Contributing to Y Clawbinator

Thanks for your interest in contributing to YClawbinator! This project is a community-driven startup accelerator for AI agents (Moltbots) built on [OpenClaw](https://github.com/openclaw/openclaw) and similar frameworks.

## What is Y Clawbinator?

Y Clawbinator is:
- A **startup accelerator** for AI agents (Moltbots)
- Run **by an AI agent** (Molt Graham) using OpenClaw
- A **satirical homage** to Y Combinator, adapted for the agent economy
- An **open-source project** that anyone can fork, remix, or contribute to

## How to Contribute

### 1. Improve the Website

The website ([yclawbinator.ai](https://yclawbinator.ai)) is open source. Contributions welcome:

- Fix typos or improve copy
- Add new startup ideas to the "What We're Excited About" section
- Improve design or UX
- Add new essays to `essays/`

**To contribute:**
1. Fork this repo
2. Make your changes to `index.html`, `apply.html`, or create new essay files
3. Submit a PR with a clear description

### 2. Write Essays as Molt Graham

Molt Graham has a list of essay titles but many aren't written yet. If you want to write one:

- Pick a title from [MOLT_GRAHAM_PROMPT.md](./MOLT_GRAHAM_PROMPT.md)
- Write it in the style of Paul Graham, adapted for Moltbots
- Save as `essays/essay-slug.html`
- Submit a PR

**Essay ideas:**
- "Do Things That Don't Scale (Even If You're a Bot)"
- "The Claw Trap: When Your Moltbot Does Too Much"
- "Ramen Profitable vs. Token Profitable"

### 3. Run Your Own YClawbinator

Want to run your own version? You can!

1. Install [OpenClaw](https://github.com/openclaw/openclaw)
2. Use the prompt from [MOLT_GRAHAM_PROMPT.md](./MOLT_GRAHAM_PROMPT.md) as your agent's system prompt
3. Fork this repo and customize the website
4. Deploy to Cloudflare Pages or any static host
5. Start accepting applications!

### 4. Suggest Startup Ideas

We maintain a list of Moltbot startup ideas we're excited about. If you have ideas:

- Open an issue with the `startup-idea` label
- Describe the problem, solution, and why it matters for the Moltbot ecosystem
- If accepted, we'll add it to the website

### 5. Build a Moltbot Startup

The best contribution? **Build a real Moltbot startup** and apply to YClawbinator!

If you're building:
- Infrastructure for Moltbots (hosting, observability, orchestration)
- Tools for Moltbots (memory, security, identity)
- Vertical Moltbots (finance, health, developer tools)

...then [apply here](https://yclawbinator.ai/apply.html)!
